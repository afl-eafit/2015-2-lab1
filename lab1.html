Programming lab 1: Simulating alternating finite automata

Submission deadline: Friday, August 21, 2015 (11:59 p.m.)

TA: Alejandro Gómez-Londoño

Introduction

The aim of this programming lab is to simulate an alternating finite automaton using Haskell.

Assignment (80%)

  • Download the GitHub repository 2015-2-lab1.
    • The module Data.Automata.AFA implements an alternating finite automata.
    • The module Data.Automata.AFA.Examples gives an example of the data type AFA.
  • (70%) Using the module Data.Automata.AFA, define a function 
              accepts :: AFA state symbol -> [symbol] -> Bool
    that tests whether an alternating finite automaton accepts or rejects an input word.
    • Using the module Data.Automata.AFA.Examples, for example, accepts m [A, A] = False.
  • (10%) Annotate your source code using Haddock. Documentation should explain your solution for the assignment.

Clean code (10%)

Before submitting your code, clean it up! Clean code:
  • Does not have long lines (at most 80 columns).
  • Has a consistent layout.
  • Has type signatures for everything.
  • Has good comments.
  • Has no junk (unused code, commented code, unnecessary code).
  • Has no overly complicated function definitions.
  • Does not contain any repetitive code.

Submission (10%)

Submission is electronic. Send an email to asicard(at)eafit(dot)edu(dot)co and agomezl(at)eafit(dot)edu(dot)co and attach a compressed file (.zip or .tar.gz) with your solution.

Your submission has to include the following:

  • SimAFA.hs, a Haskell module file with your solution for the assignment.
  • README (or README.md), a file containing at least the following information.
    • The name of your program.
    • Your name(s).
    • A general description of your program.
    • Information on how to use your program.

Notes

  • The programming lab may either be solved on your own, or jointly with one other student taking the course.
  • You are to use the Haskell Platform 2014.2.0.0 or GHCi ≥ 7.8.3.
  • You can add class constraints to the type signature of accepts if you need them.
